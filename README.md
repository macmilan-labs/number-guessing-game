# Number Guessing Game 🎮

A terminal-based Python game where the player tries to guess a randomly generated number within a limited number of attempts.

This project was built as part of learning Python fundamentals and improving programming logic through hands-on projects.

---

## Features

- Three difficulty levels:
  - Easy (1–50, 10 attempts)
  - Medium (1–100, 7 attempts)
  - Hard (1–200, 5 attempts)
- Input validation (prevents crashes on invalid input)
- Score tracking across multiple games
- Replay system
- Clean modular code using functions

---

## How the Game Works

1. The player chooses a difficulty level.
2. The program generates a secret number within the chosen range.
3. The player tries to guess the number.
4. After each guess, the program tells whether the guess was:
   - Too high
   - Too low
5. The player wins if the correct number is guessed before attempts run out.
